## Краткая информация

### Дары Ликов

- **Использование:** один раз за игровую встречу
- **Стоимость:** 1 пункт тьмы ведущему
- **Как получить:** определяются при создании персонажа случайным образом

### Достоинства команды

- **Использование:** доступны всем членам команды
- **Выбор:** одно достоинство на всю команду в начале игры
- **Стоимость:** зависит от конкретного достоинства

### Личные достоинства

- **Стоимость:** 5 пунктов опыта за достоинство
- **Приобретение:** может быть куплено по ходу игры
- **Количество:** персонаж может иметь неограниченное количество различных достоинств

### Стигмы пасынков

- **Тип:** врождённые, не могут быть изменены
- **Доступность:** только для персонажей-пасынков
- **Стоимость:** определяется при создании персонажа

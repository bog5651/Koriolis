# Достоинства команды

В начале игры вы должны выбрать одно достоинство, соответствующее амплуа вашей команды. Это достоинство доступно каждому из героев, но, если не сказано обратного, его эффект распространяется только на персонажа, который им воспользуется.

## Вольные торговцы {#merchants}

### Нюх на бирры {#nukh-na-birry}

Торгуясь, вы получаете **+2** при проверках влияния. Использовать это достоинство можно только один раз за игровую встречу, причём один раз на всю команду, а не по разу на каждого из героев.

### Ты — мне, я — тебе {#ty-mne-ya-tebe}

У вас просто талант находить самых коррумпированных чиновников, таможенников и стражников — вы всегда получаете **+2** при проверках влияния, когда пытаетесь кого‑нибудь подкупить.

### Кратчайший путь {#kratchajshij-put}

Вы находите самый быстрый путь к цели. В результате вы либо сокращаете время путешествия вдвое (см. стр. 144), либо избегаете одной нежелательной случайной встречи в пути. Ведущий при этом получает **1 пункт тьмы**. Использовать это достоинство можно только один раз за путешествие.

## Наёмники {#mercenaries}

### Огонь! {#ogon}

Вы получаете **+2** при проверках стрельбы, но только если все герои в этом раунде атакуют одну и ту же цель.

### В атаку! {#v-ataku}

Вы получаете **+2** при проверках ближнего боя, но только если все герои в этом раунде потратят хотя бы один пункт действия (быстрое действие) на перемещение. Этот модификатор действует только один раунд.

### Не зевать! {#ne-zevat}

Вы всегда зорко следите за обстановкой и получаете **+2** при проверках наблюдательности, когда нужно выяснить, заметили вы вражескую засаду или нет.

## Агенты {#agents}

### Сто друзей {#sto-druzej}

Оказавшись в новом месте, вы мигом обзаводитесь массой полезных знакомств. Новоявленные знакомые могут дать вам убежище, одолжить снаряжение или поручиться за вас перед нужными людьми. Ведущий при этом получает **1 пункт тьмы**. Использовать это достоинство можно только один раз за игровую встречу, причём один раз на всю команду, а не по разу на каждого из героев.

### Гильдия убийц {#gildiya-ubijc}

Атакуя противника, который не знает о вашем присутствии, вместо ближнего боя вы можете использовать скрытность.

### Танцоры Алама {#tancory-alama}

Пытаясь произвести хорошее впечатление, вместо влияния вы можете использовать проворство.

## Исследователи {#explorers}

### Бывалые путешественники {#byvalye-puteshestvenniki}

Пытаясь понять обычаи того или иного общества или какой‑либо группы людей, вместо мудрости вы можете использовать влияние.

### Остаться в живых {#ostat-v-zhivyh}

Вся ваша команда может спастись от любой опасности «природного» происхождения, будь то лесной пожар, песчаная буря, обвал или, скажем, взрывная декомпрессия. Ведущий при этом получает **1 пункт тьмы**. Использовать это достоинство можно только один раз за игровую встречу, причём один раз на всю команду, а не по разу на каждого из героев.

### Искатели истины {#iskately-istiny}

Вы можете без всяких проверок получить от ведущего важную информацию или улику. Информация должна быть достаточно ценной, чтобы позволить персонажам обойти или преодолеть какое‑нибудь небольшое предусмотренное сценарием приключения препятствие или испытание. Ведущий при этом получает **1 пункт тьмы**. Использовать это достоинство можно только один раз за игровую встречу, причём один раз на всю команду, а не по разу на каждого из героев.

## Пилигримы {#pilgrims}

### Шутники {#shutniki}

Вы можете выпутаться из неприятностей, развеселив публику и обратив всё происходящее в шутку. Ведущий при этом получает **1 пункт тьмы**. Использовать это достоинство можно только один раз за игровую встречу, причём один раз на всю команду, а не по разу на каждого из героев.

### Милость Ликов {#milost-likov}

Милость Ликов снисходит на тех, кто отчаянно в ней нуждается, — вы можете отменить эффект неприятности, которую ведущий только что призвал на ваши головы при помощи пунктов тьмы (см. главу 3). Использовать это достоинство можно только один раз за игровую встречу, причём один раз на всю команду, а не по разу на каждого из героев.

### Без бирра за душой {#bez-birra-za-dushoj}

Прибыв на новое место, вы можете заработать на жизнь своим ремеслом или публичными представлениями. Если вы выиграете во встречной проверке проворства против влияния местного представителя власти (или щедрого мецената), вы бесплатно обеспечите себе спартанские условия проживания (см. стр. 107) и запасные части для ремонта износившегося оборудования и снаряжения.
